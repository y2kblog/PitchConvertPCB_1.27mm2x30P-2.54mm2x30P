# ピッチ変換基板(1.27mm 2×30P⇔2.54mm 2×30P)

<img src="/img/ProductImage.jpg" width="300px">

## 概要
1.27mmピッチ2×30Pを2.54mmピッチ2×30P(300mil幅)に変換する基板です。

## 仕様
- ピンピッチ：1.27mm 2×30P ⇔ 2.54mm 2×30P
- 基板サイズ：10.16mm × 76.2mm
- 基板厚：1.6mm
- 基板色：緑
- 表面仕上げ：無鉛はんだ
- 両面スル―ホール
- 固定穴：70mmピッチ, φ3.2 × 2穴
